用返回值返回异常或错误代码。<!--more-->  
这个在原文中没内容。查了一下原文附带的文档，指的可能是创建一个复合对象（union），可带错误码和返回值，通过判断是否发生错误，获取返回值或错误码。  
个人觉得并不实用，因为实现这个复合对象是要花时间的（而且会引入复杂度），可以改为多返回值（用tuple），这样使用更加方便。
